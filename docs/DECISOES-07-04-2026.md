## ⚽ Novo Sistema de Atributos (3 Atributos)

### Atributos Universais
Todos os jogadores (incluindo goleiro) usam os mesmos 3 atributos:

| Atributo | Descrição | Ações Associadas |
|----------|-----------|------------------|
| **Potência** | Força física e impacto | Chute (ataque), Bloqueio (defesa) |
| **Rapidez** | Velocidade e agilidade | Drible (ataque), Desarme (defesa) |
| **Técnica** | Habilidade e precisão | Passe (ataque), Interceptação (defesa) |

### Distribuição de Pontos
- **Total**: 9 pontos
- **Obrigatório**: 1 ponto em cada atributo (3 pontos usados)
- **Livres**: 6 pontos para distribuir livremente
- **Limites**: Mínimo 1, Máximo 5 por atributo

### Mapeamento Ação → Atributo
```
Ofensivo:
- Chute → Potência
- Drible → Rapidez
- Passe → Técnica

Defensivo:
- Bloqueio (vs Chute) → Potência
- Desarme (vs Drible) → Rapidez
- Interceptação (vs Passe) → Técnica
```

### Goleiro (Mecânica Especial)
**Atributos**: Mesmos 3 (Potência, Rapidez, Técnica)  
**Distribuição**: 9 pontos (min 1, max 5)

**Mecânica de Defesa**:
1. Sistema escolhe aleatoriamente entre **Captura** ou **Espalme**
2. Cálculos:
   - **Espalme**: `d20 + floor((Potência + Rapidez) / 2)`
   - **Captura**: `d20 + floor((Potência + Técnica) / 2)`
3. **Arredondamento**: Sempre para baixo (floor)

**Efeitos**:
- **Captura bem-sucedida**: Goleiro mantém posse, pode passar para zona MI
- **Espalme bem-sucedido**: Bola sobra 50/50 para qualquer time

**Exemplo**:
```typescript
// Goleiro com:
Potência: 3
Rapidez: 2
Técnica: 4

// Se escolher Espalme:
Bônus = floor((3 + 2) / 2) = floor(2.5) = 2
Rolagem = d20 + 2

// Se escolher Captura:
Bônus = floor((3 + 4) / 2) = floor(3.5) = 3
Rolagem = d20 + 3
```
