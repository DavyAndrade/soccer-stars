# Documentação

Esta pasta contém toda a documentação do projeto Soccer Stars.

## Arquivos

- **GDD.md** - Game Design Document (mecânicas e regras do jogo)
- **CLAUDE.md** - Guia para agentes AI Claude
- **GEMINI.md** - Guia para agentes AI Gemini

## Padrão Lexical

Para manter consistência entre os documentos:

- **Conferência**: usar com acento quando em português (EAST/WEST como nomes oficiais).
- **Fase regular**: usar para o calendário de 22 rodadas dentro da conferência.
- **Rodada**: usar para cada semana/conjunto de confrontos da fase regular.
- **Temporada**: usar para o ciclo completo da carreira (fase regular + eventuais etapas finais).
- **Time**: termo principal para clube/equipe em toda a documentação.
- **Prioridade de zona**: regra de seleção de marcador/portador com aleatoriedade ponderada por zona.
